HTML

What is HTML?

HTML, as I've stated above, is basically the skeleton for any website. When you open any website on Chrome, it is HTML that displays the content there. it is usually made up of words, sentences, buttons, links, lists and so on. Simply, whatever you see apart from colors on a website, is HTML.

Elements of HTML

HTML is divided into so many elements. These elements are used to add properties such as paragraphes and links.
